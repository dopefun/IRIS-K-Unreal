# Возможности (текущая версия)

## Тренажёрный полигон-зал
- **Уровень «Управление»** — пошаговый квест на базовые манёвры: взлёт/удержание, повороты, наклоны.
- **Уровень «Камера»** — тренировка наклона камеры и изменения угла обзора (**FOV/zoom**).
- **Уровень «Режимы»** — обучение управлению в разных режимах (**Angle/Acro/Air/Velocity/Cine**).

## Профили дронов
- **MJI (FPV)** — манёвренный профиль с «живым» откликом для динамичных упражнений.
- **Point (съёмочный)** — стабилизированная модель с кинематографичной камерой для плавных проходов.

## Что доступно игроку сейчас
- Запуск и прохождение уровней на тренировочном полигоне (управление, камера, режимы).
- Пошаговые подсказки в рамках квеста уровня.
